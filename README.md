# 📄 PDF Merger using Python

A simple Python tool to merge multiple PDF files into one.

## 🚀 Features
- Merge multiple PDFs
- File existence validation
- Custom output file name

## 🛠️ Tech Used
- Python
- PyPDF2

## ▶️ How to Run
1. Install dependency:
   pip install PyPDF2

2. Run the script:
   python pdf_merger.py
